Каналы

Каналы по умолчанию безномерные, номера могут быть подключены к каналам "Универсальный" и "Для входящих". Для подключения мобильных номеров и активации многоканальных входящих звонков выберите номера. Каналы подключаются для использования только на территории Украины (кроме канала "Для входящих").

  • Канал "Универсальный" (исходящие-входящие звонки)
    Исходящие звонки. В один канал каждый месяц включена 1 тыс. минут исходящих звонков на всех мобильных GSM-операторов Украины, по истечении этого лимита исходящие звонки тарифицируются по 30 коп\мин (с НДС). За каналом не закреплены фиксированные номера операторов, и при звонках на мобильные телефоны отображаемые номера на вызываемом телефоне будут разными. Для удобства нами реализован механизм, в результате которого при обратном перезвоне на этот номер звонящий попадает непосредственно на канал, с которого исходил звонок – вне зависимости от того, через какое время происходит перезвон (через день или через год). Как показывает практика, этого вполне хватает для нормальной работы практически любой компании.
    Входящие звонки. К каналу может быть подключён любой существующий номер мобильных операторов Украины (который Вы можете заказать у нас). Каждый мобильный номер может быть сделан многоканальным (до 30 каналов и выше). Все входящие звонки бесплатные.
    Стоимость канала – 260 грн. в мес. (с НДС).
    Канал по умолчанию безномерной. По желанию клиента к каналу может быть привязан номер мобильного оператора.

    *За каждый обратный звонок на канал «Универсальный» с номера Lifecell согласно правил оператора за каждый факт звонка снимается плата за переадресацию в размере 0,22 грн.
    **Привязка номера работает только для входящих звонков – при исходящих звонках будет отображаться один из номеров нашего номерного пула.
  • Канал "Для исходящих" (исходящие звонки без перезвона)

    Исходящие звонки. В один канал каждый месяц включена 1 тыс. минут исходящих звонков на всех мобильных GSM-операторов Украины, по истечении этого лимита исходящие звонки тарифицируются по 30 коп\мин (с НДС). За каналом не закреплены фиксированные номера операторов, и при звонках на мобильные телефоны отображаемые номера на вызываемом телефоне будут разными. Обратный перезвон на эти номера отключен.

    Стоимость канала – 220 грн. в мес. (с НДС).

    Канал по умолчанию безномерной.

  • Канал "Для входящих" (только входящие звонки)

    К каналу возможно подключение любых мобильных номеров Украины – как давно существующих, так и новых (подробнее см. в разделе Номера). Каждый мобильный номер можно сделать многоканальным, т.е. подключить его к нескольким таким каналам (до 64-х и выше).

    Может быть подключён один запоминающийся входящий номер на всех операторов (который мы подбираем по заказу), например:

    (050)
    (067) 461-77-55
    (093)

    Стоимость одного канала - 160 грн. в мес.(с НДС). Канал по умолчанию безномерной, привязка номера к каналу 50 грн. в мес (Lifecell 80 грн).
    *При привязке номера Лайф согласно правил оператора за каждый факт звонка снимается плата за переадресацию в размере 0,22 грн. Для получения условий по использованию канала за пределами Украины обратитесь к нам.

    * Особенность каналов для Lifecell: поскольку наша технология основана на переадресации (а по условиям оператора Lifecell любая переадресация, даже внутри сети, является платной), стоимость каждого факта входящего звонка составляет 22 коп. Оплата за переадресации текущего месяца учитывается в счёте следующего месяца.

Каналы могут быть подключены:

  • к офисным мини- АТС
  • к колл-центрам
  • к системам Asterisk и подобным *
  • к системам Bitrix24 и подобным
  • к виртуальным АТС (Бинотел и подобным)
  • к системам колл-трекинга (Ringostat и проч.)
  • к SIP-телефону (программному либо стационарному)
  • и наверное ко всему, к чему вообще может быть подключен SIP-канал : )

* При подключении каналов к собственным системам телефонии заказчика Asterisk и подобным проводится обязательная проверка оборудования на безопасность.
